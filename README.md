# UFlow
- Write user flow
- Build diagram

## Summary
UFlow generate a user flow diagram from a document of a user flow with YAML format.

## Build
```
swift build
```

## How to use
```
uflowc -o output_directory user_flow_doc
```

See [`Login.yml`](./Examples/Login.yml).
